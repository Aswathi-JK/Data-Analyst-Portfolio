# 📊 Sales Data Analysis

## 📌 Objective

Analyze sales data using Python and Pandas to identify business insights, top-performing regions, profitable product categories, and loss-making products.

---

## 📂 Dataset

The project uses a sales dataset in CSV format containing information such as:

* Sales
* Profit
* Region
* Product Category
* Product Name

---

## 🛠️ Tools Used

* Python
* Pandas
* CSV Files
* Git & GitHub

---

## 🔍 Analysis Performed

### 1. Data Exploration

* Loaded sales dataset using Pandas
* Examined dataset structure and column information
* Reviewed sample records

### 2. Sales Analysis

* Calculated Total Sales
* Calculated Total Profit

### 3. Regional Performance Analysis

* Aggregated sales by region
* Identified the top-performing region

### 4. Product Category Analysis

* Aggregated sales by product category
* Identified the highest revenue-generating category

### 5. Loss Analysis

* Identified products with negative profit
* Extracted loss-making products for further investigation

---

## 📁 Outputs Generated

The following output files are generated by the Python script:

* `region_sales.csv`
* `category_sales.csv`
* `loss_products.csv`
