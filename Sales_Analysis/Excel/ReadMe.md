# 📊 Sales Analysis Using Excel

## Objective

Analyze sales data using Microsoft Excel to identify top-performing regions and product categories and present findings through an interactive dashboard.

## Dataset

* Sales dataset (CSV)
* Contains regional and category-wise sales information

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard Design

## Analysis Performed

### Regional Analysis

* Calculated total sales by region
* Identified the highest and lowest performing regions

### Category Analysis

* Calculated total sales by product category
* Compared category performance

### Dashboard

* Total Sales KPI
* Top Region KPI
* Top Category KPI
* Sales by Region Chart
* Category-wise Sales Chart

## Key Insights

* Total sales reached approximately 14.92M.
* West generated the highest sales revenue.
* Ontario and Prairie were strong contributing regions.
* Technology was the highest revenue-generating category.
* Office Supplies generated the lowest sales revenue.
